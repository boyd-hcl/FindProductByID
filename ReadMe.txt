This project contains a web application that displays a form to the user, 
takes in a product ID and displays any matches on form submission.

To set up a system for this project, 
1) initialize database products using mysql under database schema "products".
2)  Create and populate a table named "products" in database "products"
	containing fields product_id (int), name, and price.
3) deploy to Apache Tomcat using Eclipse IDE by opening WebContent/WEB-INF/index.html 
	and pressing ctrl+f11 after adding apache tomcat as the server for this project.
